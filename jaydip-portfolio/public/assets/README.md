# Assets Directory

Place all your images here following this structure:

```
assets/
├── hero-image.jpg              # Main hero image for homepage
├── about/
│   └── jaydip.jpg             # About page profile image
├── gallery/
│   ├── wedding-1.jpg          # Wedding photos
│   ├── wedding-2.jpg
│   ├── pre-wedding-1.jpg      # Pre-wedding photos
│   ├── engagement-1.jpg       # Engagement photos
│   ├── family-1.jpg           # Family portraits
│   ├── family-2.jpg
│   ├── maternity-1.jpg        # Maternity photos
│   ├── portrait-1.jpg         # Portrait photos
│   ├── lifestyle-1.jpg        # Lifestyle photos
│   ├── lifestyle-2.jpg
│   ├── model-1.jpg            # Model photos
│   └── model-2.jpg
├── process/
│   ├── step-1.jpg             # Process step 1 (Consult)
│   ├── step-2.jpg             # Process step 2 (Plan)
│   ├── step-3.jpg             # Process step 3 (Shoot)
│   └── step-4.jpg             # Process step 4 (Deliver)
└── services/
    ├── wedding.jpg            # Service thumbnail - Events
    ├── portrait.jpg           # Service thumbnail - Portraits
    └── lifestyle.jpg          # Service thumbnail - Lifestyle
```

## Image Requirements

- **Format**: JPEG or WebP (JPEG recommended for compatibility)
- **Optimization**: Compress images before adding (use tools like TinyPNG)
- **Hero Image**: 1920x1080px minimum
- **Gallery Images**: 1200x800px minimum
- **Thumbnails**: 600x400px minimum
- **Process Images**: 800x600px minimum

## How to Extract Images from PDF

1. Open PDF in Adobe Acrobat or online tool
2. Use "Export Images" feature
3. Save images with descriptive names
4. Organize in folders as shown above
5. Optimize images before uploading

## Placeholder Images

If you don't have images ready, the site will show placeholder blur effects. Replace these with actual images for best results.
